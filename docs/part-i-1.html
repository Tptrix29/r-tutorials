4 Part I

  1. Print your working directory to show that you are located in an appropriate place on your device.

  2. (Review) Create a character vector called numbers, with the numbers 1 to 5 and one NA. Then, use the pipe operator to convert this to a numeric vector, omit any NA’s, and find the mean. Do not save the mean (i.e. do not use “<-” to save the operations on the numbers variable).

  3. (Review) Save the diamonds.csv file from Courseworks into a data folder in your current working directory. Load in this data frame and save it as dat_diamond.

  4. Remove the first column using one of the functions from class. Make sure this column is gone when you view the data frame.

  5. Find three unique ways to select the carat column (do not save the column to the environment).

  6. Find the mean of the carat column (do not save).

  7. Create a variable called data_prem that only contains diamonds with a Premium cut. Also, rename the “col” variable to “color” and create a new column called “expensive” that is TRUE if the price is more than $5,000 and FALSE otherwise.

  8. Take your dat_prem data set and arrange it by price in descending order. Use the function head(5) to print only the first 5 elements.
