5 Part I: Strings

  1. Find the length of each of the following using a stringr function: ‘i’, ‘like’, ‘programming’, and NA.
str_length(c("i", "like", "programming", NA))
## [1]  1  4 11 NA
  1. Write code that returns a logical (TRUE, FALSE) vector indicating whether ‘apple’, ‘banana’, ‘kiwi’ and ‘pineapple’ each contain the letter ‘a’.
fruit <- c("apple", "banana", "kiwi", "pineapple")
str_detect(fruit, "a")
## [1]  TRUE  TRUE FALSE  TRUE
  1. Write code that returns a logical (TRUE, FALSE) vector indicating whether ‘apple’, ‘banana’, ‘kiwi’ and ‘pineapple’ each contain any vowels, a,e,i,o,u.
str_detect(fruit, "[aeiou]")
## [1] TRUE TRUE TRUE TRUE
  1. Write code that replaces the first vowel in the given ‘fruits’ variable with a ‘-’.
fruits <- c("one apple", "two pears", "three bananas")
str_replace(fruits, "[aeiou]", "-")
## [1] "-ne apple"     "tw- pears"     "thr-e bananas"
  1. Write code that replaces all vowels in the given ‘fruits’ variable with a ‘-’.
str_replace_all(fruits, "[aeiou]", "-")
## [1] "-n- -ppl-"     "tw- p--rs"     "thr-- b-n-n-s"
  1. letters is a built-in character vector in R that contains all 26 letters of the alphabet. Write a single line of code to get the following output:

“a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z”

# Method 1
str_flatten(letters, collapse = ", ")
## [1] "a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z"
# Method 2
str_c(letters, collapse = ", ")
## [1] "a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z"
  1. Write a single line of code using a stringr function to get the following output:
