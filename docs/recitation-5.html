Today you learned…

  • String and factor manipulation using stringr and forcats

Now we’ll practice using the functions we learned in class, and we’ll also get some experience with learning about functions from a written resource rather than from an instructor!

Load any necessary packages in your first chunk of code.

Now, with your partner, please work out the following exercises sequentially. There may be more than one way to code the exercises.

5.1 Part I: Strings

  1. Find the length of each of the following using a stringr function: ‘i’, ‘like’, ‘programming’, and NA.
str_length(c("i", "like", "programming", NA))
## [1]  1  4 11 NA
  1. Write code that returns a logical (TRUE, FALSE) vector indicating whether ‘apple’, ‘banana’, ‘kiwi’ and ‘pineapple’ each contain the letter ‘a’.
fruit <- c("apple", "banana", "kiwi", "pineapple")
str_detect(fruit, "a")
## [1]  TRUE  TRUE FALSE  TRUE
  1. Write code that returns a logical (TRUE, FALSE) vector indicating whether ‘apple’, ‘banana’, ‘kiwi’ and ‘pineapple’ each contain any vowels, a,e,i,o,u.
str_detect(fruit, "[aeiou]")
## [1] TRUE TRUE TRUE TRUE
  1. Write code that replaces the first vowel in the given ‘fruits’ variable with a ‘-’.
fruits <- c("one apple", "two pears", "three bananas")
str_replace(fruits, "[aeiou]", "-")
## [1] "-ne apple"     "tw- pears"     "thr-e bananas"
  1. Write code that replaces all vowels in the given ‘fruits’ variable with a ‘-’.
str_replace_all(fruits, "[aeiou]", "-")
## [1] "-n- -ppl-"     "tw- p--rs"     "thr-- b-n-n-s"
  1. letters is a built-in character vector in R that contains all 26 letters of the alphabet. Write a single line of code to get the following output:

“a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z”

# Method 1
str_flatten(letters, collapse = ", ")
## [1] "a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z"
# Method 2
str_c(letters, collapse = ", ")
## [1] "a, b, c, d, e, f, g, h, i, j, k, l, m, n, o, p, q, r, s, t, u, v, w, x, y, z"

5.2 Part II: Factors

  1. The bdays variable contains some months of the year. Apply the sort function to it. What does it do? Does it sort in a meaningful way?
bdays <- c("Dec", "Apr", "Jan", "Mar")
sort(bdays)
## [1] "Apr" "Dec" "Jan" "Mar"
  1. Create a vector called month_levels that lists all months of the year in order (i.e., Jan, Feb, …).
month_levels <- c("Jan", "Feb", "Mar", "Apr", "May", "Jun",
                   "Jul", "Aug", "Sep", "Oct", "Nov", "Dec")
  1. Create a factor variable using the bdays variable with month_levels as the levels. Now sort your new variable. What has changed?
# Method 1: base R method
bdays_factor <- factor(bdays, levels = month_levels)

# Method 2: forcats method
bdays_factor2 <- bdays %>% as_factor()
levels(bdays_factor2) <- month_levels

sort(bdays_factor)
## [1] Jan Mar Apr Dec
## Levels: Jan Feb Mar Apr May Jun Jul Aug Sep Oct Nov Dec
