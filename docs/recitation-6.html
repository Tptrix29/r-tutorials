6 Recitation 6

Today you learned…

  • Useful functions for exploratory data analysis (EDA) like case_when(), group_by(), and summarize().

Now we’ll practice using the functions we learned in class.

Load any necessary packages in your first chunk of code.

Please work out the following exercises sequentially. There may be more than one way to code the exercises.

6.1 Part I

  1. Import the Airbnb data set as bnb_data and apply clean_names().
bnb_data <- read_csv("dataset/nyc_airbnb.csv") %>% 
  janitor::clean_names()
  1. Arrange the data (and save it) in decreasing order by price.
bnb_data <- bnb_data %>% arrange(desc(price))
  1. Use if_else() to create a new variable called price_bin that equals 1 if the Airbnb is no less than average, and 0 if it is cheaper than average.
bnb_data <- bnb_data %>% 
  mutate(price_bin = ifelse(price >= mean(price), 1, 0))
  1. Use case_when() to do the same thing.
bnb_data <- bnb_data %>% 
  mutate(price_bin = case_when(price >= mean(price) ~ 1,
                               price < mean(price) ~ 0)
  )

# another version
bnb_data <- bnb_data %>% 
  mutate(price_bin = case_when(price > mean(price) ~ 1,
                               TRUE ~ 0)
  )
  1. Find the mean, median, sd, min, and max of the price variable for each neighborhood (neighborhood_group). Use group_by() and summarize().
bnb_data %>% 
  group_by(neighbourhood_group) %>% 
  summarize(mean = mean(price),
            sd = sd(price),
            min = min(price),
            max = max(price))
